# Модуль синтеза речи голосового ассистента

## Назначение
Модуль синтеза речи отвечает за преобразование текстовых ответов ассистента в аудиоформат, позволяя системе вести голосовой диалог с пользователем.

## Используемые технологии
- **Google Text-to-Speech (TTS)** — облачный сервис генерации речи от Google.
- **Amazon Polly** — платформа от AWS с поддержкой различных голосов и языков.
- **OpenAI TTS** (при доступности) — модель синтеза речи на базе GPT.

## Процесс преобразования текста в речь
1. **Приём текста**: модуль получает строку текста от ядра ассистента.
2. **Выбор сервиса**: в зависимости от настроек выбирается провайдер TTS (Google, Amazon и т.д.).
3. **Запрос к API**: текст отправляется на сервер выбранного сервиса.
4. **Генерация речи**: сервис возвращает аудиоданные в выбранном формате.
5. **Воспроизведение**: аудиофайл передаётся клиенту для проигрывания.

## Пример входных и выходных данных
- **Вход** (текст): `"Здравствуйте! Чем могу помочь?"`
- **Выход** (аудиофайл): `hello.mp3`

## Настройки голоса
- **Язык**: `ru-RU`, `en-US`, `de-DE` и т.д.
- **Тон**: нейтральный, дружелюбный, серьёзный.
- **Скорость речи**: от медленной до быстрой.
- **Тембр/голос**: выбор из предустановленных вариантов (мужской/женский).

## Поддерживаемые языки
| Язык | Пример кода | Поддерживаемые сервисы |
|------|-------------|-------------------------|
| Русский | `ru-RU` | Google TTS, Amazon Polly |
| Английский | `en-US` | Google TTS, Amazon Polly, OpenAI TTS |
| Немецкий | `de-DE` | Amazon Polly |

## Ограничения
- Качество и доступность голосов зависят от выбранного провайдера.
- Некоторые языки могут поддерживаться не всеми сервисами.
- Ограничения по скорости запросов и стоимости использования API.

## Пример кода для вызова API
```javascript
import fs from 'fs';
import fetch from 'node-fetch';

async function synthesize(text) {
  const response = await fetch('https://api.example-tts.com/v1/speak', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${process.env.TTS_API_KEY}`
    },
    body: JSON.stringify({
      input: text,
      voice: {
        languageCode: 'ru-RU',
        name: 'ru-RU-Standard-A'
      },
      audioConfig: {
        audioEncoding: 'MP3'
      }
    })
  });

  const arrayBuffer = await response.arrayBuffer();
  fs.writeFileSync('output.mp3', Buffer.from(arrayBuffer));
  console.log('Аудиофайл сохранён: output.mp3');
}

synthesize('Привет! Я ваш голосовой ассистент.');
```
